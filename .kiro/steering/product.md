# Hotel Reservation Management System

A centralized hotel reservation management web application that provides comprehensive property and booking management capabilities.

## Core Features

- **Property Management**: Hotel and room management with capacity tracking
- **Reservation System**: Complete booking lifecycle with conflict detection and availability checking
- **Real-time Updates**: SignalR-powered live notifications and calendar updates
- **External Integration**: Booking.com API integration for channel management
- **Reporting & Analytics**: Comprehensive reporting with dashboard insights
- **Role-based Access**: JWT authentication with role-based authorization

## Business Domain

The system manages the complete hotel reservation workflow from property setup through guest check-out, with emphasis on:
- Preventing double bookings through robust conflict detection
- Real-time availability management
- Integration with external booking channels
- Performance optimization for high-concurrency scenarios
- Comprehensive audit trails and reporting

## Target Users

- **Hotel Managers**: Property and reservation oversight
- **Front Desk Staff**: Daily operations and guest management  
- **System Administrators**: Configuration and user management
- **External Systems**: API integration for booking channels
